Data sourced from NYT & covidtracking
https://github.com/nytimes/covid-19-data
https://covidtracking.com/data

CovidTracking Data Definitions: https://covidtracking.com/about-data/data-definitions
date - date of observation
state - US state 2 digit code
positive - number of tests with positive results
negative - number of tests with negative results
pending - number of test with pending results
hospitalizedCurrently - Covid patients in hospital at date of recording
hospitalizedCumulative - Cumulative total of Covid hospitalizations
death - number of deaths
total - total number of tests
inIcuCurrently - Covid patients admitted into ICU at time of recording.
inIcuCumulative	- Total number of individuals who have ever been hospitalized in the Intensive Care Unit with COVID-19
onVentilatorCurrently - Individuals who are currently hospitalized under advanced ventilation with COVID-19.
onVentilatorCumulative - Total number of individuals who have ever been hospitalized under advanced ventilation with COVID-19
recovered - Total number of people that are identified as recovered from COVID-19
dateChecked - Date/Time the data was checked in.
death - Total fatalities with confirmed OR probable COVID-19 case diagnosis
total - Covid Case count cumulative total
totalTestResults - Total number of tests performed
deathIncrease - Daily increase in Covid Deaths
hospitalizedIncrease - Daily increase in number of COVID-19 patients hospitalized
negativeIncrease - Daily increase in negative tests
positiveIncrease - Daily increase in positive tests
hash - Unique ID
totalTestResultsIncrease - Combined increase in test results processed.
